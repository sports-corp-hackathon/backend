backend
=======

Java backend for the sports corp hackathon application. Currently deployed on EC2. Uses DynamoDB and SNS. 

Requisities:  
`java 1.7`
`export AWS_ACCESS_KEY="your access key here"`  
`export AWS_SECRET_KEY="your secret key here"`

Running:  
`mvn clean compile`  
`mvn exec:java`
